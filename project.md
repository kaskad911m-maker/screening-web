# Цифровой скрининг личности — Паспорт проекта

## Что это

Онлайн-сервис психологического скрининга личности от практикующего психолога Марины Филимоновой. Человек проходит диагностику и получает персональный разбор своего психотипа, нейробаланса и внутренних ресурсов.

## Для кого

- Женщины 28–50 лет, которые чувствуют внутренний разлад или «что-то не так»
- Те, кто хочет понять себя глубже, но не готов идти на длинную терапию
- Люди, ищущие ясность: кто я, почему я так реагирую, как мне стать эффективнее
- Предприниматели и специалисты, которые хотят опираться на свои сильные стороны

## Что предлагаем

- Психологический скрининг личности (онлайн-встреча 60–90 мин)
- Цифровой разбор психотипа и нейробаланса
- Персональные рекомендации по питанию, режиму, развитию
- Лендинг с описанием услуги и записью на встречу
- Квиз для диагностики психотипа (4 типа: Визионер, Архитектор, Проводник, Исследователь)

## Чего НЕ будет

- Массовых курсов и марафонов
- Автоматических отчётов без живого участия психолога
- Медицинских диагнозов
- Долгих онлайн-программ (только точечная диагностика)

## Исследование рынка

### Конкуренты

- Онлайн-психологи на сервисах (Zigmund.online, Yasno.live) — работают в формате терапии, без быстрого скрининга
- Нумерологи и астрологи — субъективный подход, нет научной базы
- Коучи — фокус на цели, не на личность
- **Наше преимущество:** сочетание психологии + нейробиологии + практического разбора за одну встречу

### Функции (MVP)

- Лендинг с описанием услуги и кнопкой записи
- Квиз-диагностика психотипа
- Страница визитки специалиста
- Форма для записи на скрининг

### UX (пользовательский опыт)

- Сайт должен создавать атмосферу доверия и спокойствия
- Минималистичный, премиальный дизайн (тёмные/глубокие цвета)
- Чёткое объяснение: что получит человек после скрининга
- Простая запись — не больше 2 шагов

### Технологии

- HTML/CSS — лендинг и визитка (уже готово)
- GitHub Pages — хостинг (подключён)
- Google AI Studio — квиз-диагностика
- VS Code + Claude Code — разработка

### Монетизация

- Основная: личный скрининг (разовая консультация)
- Дополнительная: пакеты встреч, авторские методики
- Перспектива: онлайн-программа на основе психотипа

## Критический взгляд (что улучшить)

- Нет чёткого призыва к действию после квиза (куда идти человеку?)
- Форма записи пока не автоматизирована
- Нет социальных доказательств (отзывы, количество клиентов) прямо на первом экране
- Стоимость услуги не указана — это может отпугивать или наоборот интриговать (решить осознанно)
- Нужен раздел «Как проходит скрининг» — пошаговый процесс снижает тревогу клиента
