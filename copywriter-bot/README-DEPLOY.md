# Бот 24/7 — когда ноутбук выключен

Пока бот запущен только как **`npm start` на твоём ПК**, он **не работает**, когда ноутбук спит или выключен.  
Чтобы отвечал **с телефона в любое время**, нужно запустить **тот же код** на сервере в интернете и передать туда **те же переменные** из `.env` (токен Telegram + ключ OpenRouter).

Нейросеть **уже привязана** через `.env` (`CHAT_API_URL`, `CHAT_API_KEY`, `CHAT_MODEL`) — на сервере ты укажешь **те же три строки** в настройках хостинга, не в файле.

---

## Ссылкой на бота делиться можно

Формат: **`https://t.me/твой_username_бота`**  
(тот username, что задавала в BotFather, с `@` в начале в Telegram, в ссылке без `@`).

Это **не** ссылка на твой ноутбук — это адрес бота в Telegram. Работать будет, когда бот **крутится на сервере**.

---

## Вариант A — Railway (удобно для Node)

1. Зарегистрируйся на **https://railway.app** (можно через GitHub).  
2. **New Project** → **Deploy from GitHub repo** (репозиторий с `my-first-project`).  
3. В настройках сервиса укажи **Root Directory / Watch Path:** **`copywriter-bot`**  
   (чтобы собиралась только папка бота, не весь сайт).  
4. **Variables** (переменные окружения) — добавь **вручную** (значения скопируй из своего `.env`, **не** коммить `.env` в Git):

   - `TELEGRAM_BOT_TOKEN`  
   - `CHAT_API_URL` = `https://openrouter.ai/api/v1/chat/completions`  
   - `CHAT_API_KEY`  
   - `CHAT_MODEL` = `openai/gpt-4o-mini`  
   - опционально `USE_MULTISTEP` = `0` или `1`  
   - опционально тренды и картинки: `TAVILY_API_KEY`, `TAVILY_TOPIC`, `TAVILY_TIME_RANGE`, `TAVILY_SOCIAL_EXTRA`, `IMAGE_MODEL`, `NICHE_HINT` — см. `.env.example`

5. **Deploy**. В логах должно быть «Бот запущен» / без ошибок.  
6. Останови **`npm start`** дома, чтобы не было **двух** экземпляров с одним токеном (Telegram так не любит).

*Если Railway попросит Dockerfile — он уже лежит в `copywriter-bot/Dockerfile`.*

---

## Вариант B — Render.com (Background Worker)

1. **https://render.com** → аккаунт, **New** → **Background Worker**.  
2. Подключи репозиторий, **Root Directory:** `copywriter-bot`.  
3. **Build command:** `npm install`  
4. **Start command:** `npm start`  
5. Во вкладке **Environment** добавь те же переменные, что выше.  
6. Бесплатный тариф может **засыпать** — для серьёзного канала лучше платный или Railway.

---

## Важно

- **Не заливай** файл `.env` в GitHub (он в `.gitignore`). Ключи только в панели Railway/Render.  
- После переноса в облако **один** запущенный бот на один `TELEGRAM_BOT_TOKEN`.  
- **`tone-of-voice.md`** при депое из Git попадёт на сервер вместе с репозиторием. Меняешь файл → `git push` → redeploy.
